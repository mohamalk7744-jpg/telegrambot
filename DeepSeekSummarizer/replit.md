# بوت تلخيص النصوص - Telegram Bot with DeepSeek AI

## نظرة عامة
هذا المشروع عبارة عن بوت Telegram مبني بلغة Python يستخدم DeepSeek AI API لتلخيص النصوص والملفات. يدعم البوت تلخيص النصوص المباشرة وملفات PDF وWord.

## البنية المعمارية
- **bot.py**: الملف الرئيسي للبوت الذي يحتوي على جميع الوظائف
- **Python 3.11**: لغة البرمجة المستخدمة
- **python-telegram-bot**: للتعامل مع Telegram Bot API
- **DeepSeek API**: للتلخيص الذكي للنصوص
- **PyPDF2**: لاستخراج النصوص من ملفات PDF
- **python-docx**: لاستخراج النصوص من ملفات Word

## المميزات الرئيسية
1. استقبال وتلخيص الرسائل النصية المباشرة
2. استقبال ومعالجة ملفات PDF
3. استقبال ومعالجة ملفات Word (DOCX)
4. واجهة بسيطة مع أوامر `/start` و `/help`
5. معالجة الأخطاء بشكل صحيح

## المفاتيح المطلوبة
- TELEGRAM_BOT_TOKEN: مفتاح بوت Telegram
- DEEPSEEK_API_KEY: مفتاح DeepSeek API
